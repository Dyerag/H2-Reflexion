Problem der skal løses:

Til undervisning i SQL Server i H2 skal der laves nogle sammenlignende målinger
 mellem 2 tabeller - den ene tabel med diverse indekser på og den anden uden.
Tabellerne skal kunne BCP loades og en database oprettes fra SQLCMD.....



Lav et script der kan kaldes fra kommandolinien med et valgfrit antal rækker af typen
use master 
use master 
if exists (select * from sys.databases where name = 'DBLoad')
begin
	drop database DBLoad
end
create database DBLoad
go
use DBLoad
go
Create table Dummy
(
	idex int primary key,
	Dummy1 varchar(100) not null,
	Dummy2 varchar(100) not null,
	Kommentar varchar(100) not null,
	Dato date not null
)
go
Create table DummyUdenIndex
(
	idex int,
	Dummy1 varchar(100) not null,
	Dummy2 varchar(100) not null,
	Kommentar varchar(100) not null,
	Dato date not null
)
